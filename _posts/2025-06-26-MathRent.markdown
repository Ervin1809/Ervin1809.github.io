## ğŸ’» Struktur Proyek

### ğŸ—ï¸ **Struktur Direktori**

**Frontend (React + TailwindCSS):**
- `DashboardDesign/src/` - Kode sumber utama
  - `components/` - Komponen UI yang dapat digunakan kembali
  - `assets/` - Gaya dan berkas statis
  - `services/` - Integrasi API
  - `utils/` - Fungsi pembantu

**Backend (FastAPI + Python):**
- `FastAPI/app/` - Aplikasi backend
  - `routers/` - Titik akhir API
  - `crud/` - Operasi basis data
  - `models.py` - Model basis data
  - `auth.py` - Logika autentikasi### ğŸ”§ **Titik Akhir API**

**Autentikasi**
- `POST /auth/register` - Pendaftaran mahasiswa
- `POST /auth/login` - Masuk pengguna
- `POST /auth/logout` - Keluar pengguna

**Rute Mahasiswa**
- `GET /student/items` - Mendapatkan barang tersedia
- `GET /student/items/{id}` - Mendapatkan detail barang
- `POST /student/rentals` - Membuat permintaan peminjaman
- `GET /student/rentals/history` - Mendapatkan riwayat peminjaman

**Rute Staf**
- `GET /staff/items` - Mendapatkan semua barang (CRUD)
- `POST /staff/items` - Membuat barang baru
- `PUT /staff/items/{id}` - Memperbarui barang
- `DELETE /staff/items/{id}` - Menghapus barang
- `GET /staff/rentals` - Mendapatkan semua permintaan peminjaman
- `PUT /staff/rentals/{id}/approve` - Menyetujui peminjaman
- `PUT /staff/rentals/{id}/return` - Mengonfirmasi pengembalian
